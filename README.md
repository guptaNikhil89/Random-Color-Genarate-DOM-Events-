# 🎨 Random Color Generator

A simple JavaScript project that generates a random color when the user clicks a button. The generated color is displayed on the screen along with its RGB value.

## 🚀 Features

* Generate a random color on button click
* Display the RGB color value
* Change background color dynamically
* Simple and beginner-friendly JavaScript logic

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript

project-folder

```

## ▶️ How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Open **index.html** in your browser.

## 💡 How It Works

1. A button click triggers a JavaScript function.
2. The function generates three random numbers between **0–255**.
3. These numbers form an **RGB color**.
4. The background color of the page changes to the generated color.

Example:

```
rgb(124, 45, 200)
```
