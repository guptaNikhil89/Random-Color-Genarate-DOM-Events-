# Random RGB Color Generator 🎨

This is a beginner-friendly JavaScript project that generates a random color every time a button is clicked. It demonstrates the use of DOM events and CSS manipulation.

### ✨ Features
* Generates a random RGB color code (e.g., `rgb(30, 208, 104)`).
* Updates the heading text with the current color code.
* Changes the background color of a specific `div` dynamically.

### 🛠️ Technologies Used
* HTML5
* CSS3
* JavaScript (ES6+)

### 🚀 How to Run
1. Clone this repository.
2. Open `index.html` in your browser.
3. Click the "Generate color" button!
